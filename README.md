## Возможности

- Пресеты **GOD**, **999+**, **HUMAN** и **CUSTOM** — свои WPM, accuracy и consistency
- Попап: тёмная тема, переключатель, синхронизация через `chrome.storage.sync`
- Горячие клавиши `Alt+1…4` и стоп `Alt+0` прямо на странице теста

---

## Установка

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/DenisZhukovWatchDemoEgoistMentality/MonkeytypeGOD.git
   cd MonkeytypeGOD
   ```
2. Открыть `chrome://extensions` (или `edge://extensions`)
3. Включить **Режим разработчика**
4. **Загрузить распакованное** — папка с `manifest.json`

## Режимы

| Режим | WPM | Accuracy | Комментарий |
|--------|-----|----------|-------------|
| &#128293; GOD | 150 | 100% | Максимальный темп |
| &#128142; 999+ | 120 | 99% | Сильный результат |
| &#128100; HUMAN | 80 | 96% | Спокойный ритм |
| &#9881;&#65039; CUSTOM | из попапа | из попапа | Свои значения |

---

## Горячие клавиши

На **monkeytype.com** (в фокусе вкладка с тестом):

| Сочетание | Режим |
|-----------|--------|
| `Alt` + `1` | &#128293; GOD |
| `Alt` + `2` | &#128142; 999+ |
| `Alt` + `3` | &#128100; HUMAN |
| `Alt` + `4` | &#9881;&#65039; CUSTOM (из попапа) |
| `Alt` + `0` | &#9209;&#65039; стоп |

Сочетания из `manifest.json` можно переназначить: **Расширения → Комбинации клавиш**.
